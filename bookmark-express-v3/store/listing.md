## Store listing tab

**Item name** (max 75)
```
Bookmark Express
```

**Summary** (max 132)
```
Instant search for your Chrome bookmarks. Open the popup, start typing, and hit Enter. Fast, keyboard-driven, no clutter.
```

**Description**
```
Bookmark Express is the fastest way to get to a bookmark.

Press Ctrl+Shift+K (or click the toolbar icon), start typing, and your
bookmarks filter instantly as you go. Use the arrow keys to move through the
results and Enter to open — you never have to touch the mouse.

FEATURES
• Instant search — results filter the moment you type, matching both the title
  and the URL of every bookmark.
• Keyboard-first — launch with a shortcut, navigate with ↑ / ↓, open with Enter.
• Smart ranking — the bookmarks you open most often rise to the top.
• Folder context — every result shows the folder path it lives in, so you can
  tell similar bookmarks apart at a glance.
• Match highlighting — your search terms are highlighted in each result.
• Lightweight and fast — a tiny popup that opens instantly, with no accounts,
  no tracking, and no clutter.

PRIVACY
Bookmark Express does not collect, transmit, or sell any of your data. Your
bookmarks never leave your device, and the only thing it stores — a local count
of how often you open each bookmark — stays in your browser.

Tip: you can change the keyboard shortcut at chrome://extensions/shortcuts.
```

**Category:** Productivity

**Language:** English (United States)

## Privacy tab

**Single purpose**
```
Bookmark Express lets you instantly search your Chrome bookmarks from a keyboard-driven popup and open them.
```

**Permission justifications**

- `bookmarks`
```
Used to read the user's bookmarks so they can be searched and opened from the popup. Bookmark data is only read locally and is never transmitted.
```

- `favicon`
```
Used to display each bookmarked site's favicon next to its result via Chrome's built-in favicon service.
```

- `storage`
```
Used to store a local, per-bookmark count of how many times the user has opened each bookmark, so that frequently used bookmarks can be ranked higher in the results. This data stays on the device.
```

**Are you using remote code?** → No, I am not using remote code.

**Data usage / disclosures:**
- Does your extension collect or use user data? → It does **not** collect user
  data. Do not check any data-type boxes.
- Certify the three data-usage compliance checkboxes (data is not sold, not used
  for unrelated purposes, not used for creditworthiness/lending).

**Privacy policy URL:** host `privacy-policy.md` (e.g. on alexcharland.com) and
paste its URL. Recommended even though no data is collected, because the
`bookmarks` permission often triggers a reviewer request for one.
